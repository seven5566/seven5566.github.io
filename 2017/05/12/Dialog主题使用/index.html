Android Dialog 主题使用

发表于 2017-05-12

当我们需要只显示一个对话框,而不显示一个全屏的Activity,然后在Activity中显示一个对话框,有两种方式可以完成此需求:

  • 将Activity的主题设置为Dialog属性,应该会产生的布局为Dialog,重新写布局即可
  • 将Activity的主题的属性设置为透明的,也就是Activity的属性设置为透明,然后再透明的Activity上显示一个对话框即可.
